# component

组件包 放置各种组件

## 组件类型

* db_service
    * Database 组件
    * db_xxx 是特定的 组件服务 和 配置文件 中 配置对应

* es_service
    * ElasticSearch 组件
    * es_xxx 是特定的 组件服务 和 配置文件 中 配置对应

* kafka_service
    * Kafka 组件
    * Kafka_xxx 是特定的 组件服务 和 配置文件 中 配置对应

* redis_service
    * Redis 组件
    * redis_xxx 是特定的 组件服务 和 配置文件 中 配置对应

* zk_service
    * Zookeeper 组件
    * zk_xxx 是特定的 组件服务 和 配置文件 中 配置对应

* mongodb_service
    * MongoDB 组件
    * mongodb_xxx 是特定的 组件服务 和 配置文件 中 配置对应
